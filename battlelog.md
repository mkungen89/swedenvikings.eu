# Battlelog System - Guide

## Översikt

Battlelog-systemet är inspirerat av Battlefield's battlelog och anpassat för Arma Reforger med stöd för olika moddar (WCS, ACE, RHS, etc.).

## Admin Panel

**Path**: `/admin/battlelog`

### Lägga till Vapen

1. Gå till Admin > Battlelog
2. Klicka på "Lägg till Vapen"
3. Fyll i formuläret med Name, Display Name, Type, Faction, Mod Source
4. Lägg till Image URL eller ladda upp bild

### Lägga till Fordon

1. Gå till Admin > Battlelog > Fordon-tab
2. Klicka på "Lägg till Fordon"
3. Fyll i formuläret

## Fraktion-val

Användare kan välja mellan NATO och RUS på sin profilsida under Battlelog-tabben.

## Moddar

- WCS, ACE, RHS, Vanilla, Custom mods
